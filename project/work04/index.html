DESIGN SYSTEM 01
-

STRATEGY

01
브랜드 아이덴티티를
확고히 정립합니다

다양한 화보 이미지들을 보유한 JEEP 브랜드의 강점을 살려 비쥬얼을 강조하는 디자인으로 진행하였습니다. 이를 통해 사용자들은 단어나 글자보다 더욱 확실하게 JEEP만의 아이덴티티를 경험합니다.

02
최신 모델 라인업에
우선순위를 부여합니다

기존의 가격과 프로모션을 우선으로 보여주는 방식은 이미 수입 SUV 브랜드 1,2위를 다투는 브랜드에게 큰 의미가 없다고 생각하여, 최신 모델 라인업들을 먼저 전달해주는 것에 우선 순위를 두었습니다.

03
사용자들에게 효율적인
접근성을 제공합니다

오랜 기간 걸쳐서 수정해온 웹사이트의 메뉴는 여러 카테고리로 쪼개져 있고 그룹화가 애매한 파트도 있었습니다. 그래서 메가메뉴, 스티키 네비게이션 등을 직관적으로 구성해 접근성을 높였습니다.

MAIN PAGE

01 GNB

웹 내의 이동성을 위해 고정형으로 설계. 마우스를 올리면 펼쳐지는 메가메뉴로 구성.

02 메인 비주얼

사용자들이 제일 먼저 눈에 담는 메인에는 최신 라인업으로 슬라이드 구성

03 빠른 시승신청

사용자의 편의를 위해 시승 예약 폼을 메인 비주얼 하단에 배치

04 메가메뉴

기존 사이트의 메뉴는 오랜 기간동안 끝에 추가하는 식으로 수정해온 탓에 사용자들이 카테고리르 파악하기 어려웠음. 그래서 크게 펼쳐지는 메가메뉴로 사용자들이 직관적으로 사이트의 구성을 파악할 수 있도록 유도

05 전체 모델 라인업

각 모델 라인업에서 대표적인 차량 모델을 360도로 보면서 대략적인 특징을 파악

06 JEEP의 기술력 소개

거친 자연속 러프한 모험을 가능케 하는 JEEP의 기술력 소개

07 프로모션

매달 다르게 진행하는 JEEP의 프로모션 소식을 모델 종류별로 확인

08 사용자 스토리

JEEP와 함께 하는 일반인 및 유명인들의 소식을 슬라이드 형식으로 소개

09 FOOTER

아이콘으로 자주 찾는 컨텐츠들을 표시하고, 사업자 정보 안내
